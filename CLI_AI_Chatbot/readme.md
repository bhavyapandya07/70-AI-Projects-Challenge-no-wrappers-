## Project 2: CLI AI Chatbot 💬

Building upon Day 1, this project upgrades our single-prompt script into a continuous, interactive Command Line Interface (CLI) chatbot.

---

What I learned today:

❏ Creating a continuous while loop for terminal interaction.

❏ Managing Conversation History (context) by appending user and assistant messages to a list, allowing the AI to remember what was said earlier in the chat.

❏ Handling basic terminal exceptions like KeyboardInterrupt (Ctrl+C) gracefully.

---

How to run this:

❏ Ensure Ollama is running (ollama run llama3.2:1b).

❏ Activate the virtual environment.

❏ use pip install ollama and then type ollama run llama3.2:1b

❏ Run python caicb.py
