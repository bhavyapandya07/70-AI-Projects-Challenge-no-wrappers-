This project is part of my 70-Day AI Challenge. It is a fully local Natural Language Processing (NLP) script that condenses long blocks of text into short, readable summaries. 

Instead of relying on paid APIs, this project downloads and runs a Transformer model directly on my local machine using my system's RAM and CPU.

## Tech Stack

❏ **Libraries:** `transformers`, `torch`

❏ **Model:** `sshleifer/distilbart-cnn-12-6` (via Hugging Face)
